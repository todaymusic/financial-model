# 📊 교육 사업 순이익 시뮬레이션 대시보드

전화영어 사업의 변수를 조정하며 순이익 영향을 실시간으로 확인하는 내부용 대시보드입니다.

## 주요 기능

- **실시간 시뮬레이션**: 슬라이더 조정 시 즉시 손익 재계산
- **강사 명단 관리**: 강사별 등급/월급/인센티브 개별 입력
- **상품 믹스 분석**: 6개 상품의 비중 조정 → 평균 단가 자동 산출
- **기간별 비교**: 1/3/6/9/12개월 누적 손익 토글
- **민감도 분석**: 12개 변수의 순이익 영향도 토네이도 차트
- **강사 가동률**: 주 근무시간 기반 최대 학생 수용 인원 자동 계산

## 모델에 반영된 사업 조건

- 수강료 가격표 (1/3/6/12개월 × 20분/40분 × 주1~5회, 할인율 0/10/15/20%)
- 강사 인센티브 (실버 3% / 골드 6% / 플래티넘 9%)
- 13th month pay (필리핀 노동법)
- 퇴직금 (연봉의 10%)
- 부가세 10%, 법인세 22%
- 파트너 채널 분배 (수강료의 20~30%)

## 사용 방법

1. 브라우저로 페이지 열기
2. 슬라이더 조정 (학생 수, 파트너 비중 등)
3. 강사 명단에서 등급/월급 수정 또는 추가
4. KPI와 손익 상세표 확인
5. "민감도 분석" 탭에서 핵심 변수 우선순위 파악

## 기술 스택

- Pure HTML / CSS / JavaScript (프레임워크 불필요)
- Chart.js (차트 라이브러리, CDN 로드)
- Tabler Icons (아이콘, CDN 로드)
- 서버/DB 불필요 — 모든 계산은 브라우저에서 수행

## 보안

- 모든 데이터는 브라우저 메모리에만 존재
- 새로고침 시 입력값 초기화 (의도된 동작)
- 외부 서버로 데이터 전송 없음
